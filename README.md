# MCT-bruteforce-key
Mifare Classic Tool Mod apk with bruteforce for the keys in NFC cards
# New features
• Stealth Mode

• Read, Emulate and save Credit Cards

• BCC calculator

• Emulate any UID from a tag

• Bruteforce key

• Save and edit the tag data you read

• Bug fixes

• Mifare Ultralight Support
